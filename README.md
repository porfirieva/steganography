**Для проверки задания №1:**
```-запустить файл "firts_part"
-ввести в консоль номер бита для подмены (1-8)
-указать номер сета для проверки
-программа завершит цикл после проверки 5 файлов (те, что использовались далее для исследовательской части)
-результат сохранен в папку ./1_result/<номер сета>/converted_<название исходного изображения>_bit<номер бита>
```

**Для проверки задания №2:**
```-запустить файл "second_part"
-ввести в консоль номер бита для подмены (1-8)
-по умолчанию программа использует содержимое файла (текст ~32мб) './text.txt'
-указать путь до контейнера для встраивания
-результат сохранен в папку ./2_result/<название исходного изображения>_bit<номер бита>
-сохранить итоговое количество записанных бит для задания №3
```

**Для проверки задания №3:**
```-запустить файл "third_part"
-ввести в консоль номер бита для подмены (1-8)
-указать путь до изображения для извлечения
-указать количество бит для извлечения из задания №2
-результат сохранен в папку ./3_result/extracted_message.txt
```
